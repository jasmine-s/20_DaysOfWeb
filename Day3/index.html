CSS Animations

An animation lets an element gradually change from one style to another. You can change
as many CSS properties as you want to, as many times you want to. Keyframes hold
the styles the element will have at certain times
The @keyframes Rule
When you specify CSS styles inside the @keyframes rule, the animation will gradually change from the current style to the new style at certain times. To get an animation to work, you must bind the animation to an element.
As an alternative to using percentages, you can use from and to keywords, where: from is a starting offset of 0% to is an ending offset of 100%.
    CSS suppors different animation properties as follows
  • animation-name
  • animation-duration
  • animation-delay
  • animation-iteration-count
  • animation-direction
  • animation-timing-function
  • animation-fill-mode
  • animation
